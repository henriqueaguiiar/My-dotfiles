# My Dotfiles

Este repositório contém os meus dotfiles pessoais para configuração do ambiente de desenvolvimento.

## Arquivos incluídos

Os seguintes arquivos de configuração estão incluídos neste repositório:

- `.bashrc`: Arquivo de configuração para o shell Bash.
- `.zshrc`: Arquivo de configuração para o shell Zsh.

Esses arquivos incluem configurações e alias personalizados para facilitar o desenvolvimento.
